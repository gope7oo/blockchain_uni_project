През последните години блокчейн технологията се утвърди като една от най-значимите иновации в областта на информационните технологии. Първоначално тя става известна като основа на криптовалутата Bitcoin, но впоследствие намира приложение в редица други области, включително финансови системи, логистика, управление на цифрова идентичност и интелигентни договори.

Основната идея на блокчейн е създаването на последователна верига от блокове, съдържащи информация, които са свързани помежду си посредством криптографски хешове. Тази структура осигурява висока степен на сигурност и прави промяната на вече записани данни изключително трудна.

Настоящият проект има за цел да демонстрира принципите на работа на блокчейн технологията чрез разработването на опростена симулация, реализирана с помощта на езика C++.
Блокчейн представлява децентрализирана структура от данни, в която информацията се съхранява в последователност от блокове. Всеки блок съдържа определен набор от данни, собствен хеш и хеш на предходния блок. По този начин се създава връзка между отделните блокове, която гарантира целостта на информацията.

Една от основните характеристики на блокчейн системите е тяхната неизменяемост. При промяна на информацията в даден блок неговият хеш също се променя, което води до нарушаване на връзката с останалите блокове. Именно този механизъм осигурява надеждността на технологията и я прави подходяща за приложения, изискващи висока степен на сигурност.

В реалните блокчейн системи се използват криптографски алгоритми като SHA-256, които генерират уникален цифров отпечатък на информацията. В настоящия проект е използван опростен алгоритъм, чиято цел е да демонстрира принципа на работа, без да се усложнява реализацията.
Разработеното приложение представлява конзолна програма, написана на езика C++. Системата е изградена чрез използването на обектно-ориентиран подход и се състои от два основни класа – Block и Blockchain.

Класът Block описва структурата на един блок. Той съдържа индекс, информация за транзакцията, хеш на предходния блок, собствен хеш и допълнителна стойност nonce, която участва в процеса на генериране на валиден хеш.

Класът Blockchain управлява цялата верига от блокове. Негова задача е да създаде първоначалния Genesis Block, да добавя нови блокове към веригата и да проверява нейната валидност.

При проектирането е избран опростен модел, който пресъздава основните принципи на блокчейн технологията, без да включва мрежова комуникация между възли или сложни криптографски алгоритми.
Реализацията е извършена посредством езика за програмиране C++, като са използвани стандартните библиотеки string, vector и sstream. След стартиране на програмата се създава първият блок във веригата, наречен Genesis Block. След това се добавят последователно нови блокове, съдържащи примерни транзакции.

При създаването на всеки нов блок се генерира хеш стойност, която се изчислява въз основа на съдържанието на блока, хеша на предходния блок и стойността nonce. По този начин всеки блок се свързва с предходния, което формира цялостната структура на блокчейна.

Допълнително е реализирана функция за проверка на валидността на веригата. Тази функция преминава през всички блокове и сравнява хеша на всеки предходен блок със стойността previousHash, записана в следващия блок. При несъответствие системата отчита, че целостта на блокчейна е нарушена.
Макар блокчейн да е известен предимно чрез криптовалутите, неговите приложения не се ограничават само до тях. Технологията се използва във финансовия сектор, където позволява осъществяването на сигурни транзакции без необходимост от централен посредник. Освен това блокчейн намира приложение при интелигентните договори, системите за електронно гласуване, управлението на веригите за доставки и защитата на цифровата идентичност.

Нарастващият интерес към Web 3.0 и децентрализираните приложения допълнително увеличава значението на блокчейн технологиите и ги превръща в една от основните тенденции в съвременните интернет технологии.

#include <iostream>    
#include <vector>       
#include <string>       
#include <sstream>      

using namespace std;

/*
    Цел: Симулира хеш функция
    В реалния блокчейн се използва SHA-256, но тук правим опростена версия
*/
string simpleHash(string input) {
    unsigned long hash = 5381; // начална стойност (стандартен seed)

    // обхождаме всеки символ и “смесваме” стойността
    for (char c : input) {
        hash = ((hash << 5) + hash) + c;
    }

    return to_string(hash); // връщаме като текст
}

/*
    Представя един блок в блокчейн веригата
*/
class Block {
public:
    int index;          // позиция на блока във веригата
    string data;        // информация (транзакции)
    string prevHash;    // хеш на предишния блок
    string hash;        // текущ хеш
    int nonce;          // стойност за mining (Proof of Work)

    /*
        Създава блок и автоматично го “mining”-ва
    */
    Block(int i, string d, string pHash) {
        index = i;
        data = d;
        prevHash = pHash;
        nonce = 0;

        // започваме mining процес
        hash = mineBlock(1); // difficulty = 3 (3 нули в началото)
    }

    /*
        Комбинира всички данни и изчислява hash
    */
    string calculateHash() {
        stringstream ss;

        // комбинираме всички важни данни
        ss << index << data << prevHash << nonce;

        return simpleHash(ss.str());
    }

    /*
        Симулира Proof of Work (mining)
        Търси hash, който започва с определен брой нули
    */
        string mineBlock(int difficulty) {
    cout << "Mining block #" << index << " ..." << endl;
    
    // Просто увеличаваме nonce докато hash започне с 0 (или принудително)
    for (int i = 0; i < 100000; i++) {   // лимит 100 000 опита
        hash = calculateHash();
        if (hash[0] == '0' || i > 50000) {   // по-лесно условие
            nonce = i;
            cout << "✓ Block #" << index << " mined! Nonce = " << nonce 
                 << " Hash = " << hash << endl;
            return hash;
        }
    }
    
    // Ако не намери - принудително
    hash = calculateHash();
    cout << "✓ Block #" << index << " mined (forced)! Hash = " << hash << endl;
    return hash;
    }
};

/*
    Управлява цялата верига от блокове
*/
class Blockchain {
private:
    vector<Block> chain; // списък от блокове

public:
    /*
        Създава първия блок (Genesis Block)
    */
    Blockchain() {
        chain.push_back(Block(0, "Genesis Block", "0"));
    }

    /*
        Добавя нов блок към веригата
    */
    void addBlock(string data) {
        Block prev = chain.back(); // взимаме последния блок

        // създаваме нов блок, свързан с предишния
        chain.push_back(Block(chain.size(), data, prev.hash));
    }

    /*
        Показва всички блокове в блокчейна
    */
    void showChain() {
        cout << "\n================ BLOCKCHAIN ================\n";

        for (auto &b : chain) {
            cout << "\nBlock #" << b.index << endl;
            cout << "Data       : " << b.data << endl;
            cout << "Prev Hash  : " << b.prevHash << endl;
            cout << "Hash       : " << b.hash << endl;
            cout << "Nonce      : " << b.nonce << endl;
        }

        cout << "\n===========================================\n";
    }

    /*гата е цяла (не е манипулирана)
    */
    bool isValid() {
        for (int i = 1; i < chain.size(); i++) {
            // ако връзката е счупена → блокчейнът е невалиден
            if (chain[i].prevHash != chain[i - 1].hash) {
                return false;
            }
        }

        return true;
    }
};

int main() {
    Blockchain bc; // създаваме блокчейн система
    cout << "Simple Blockchain Simulator (C++)\n";
    
    // добавяме примерни блокове (транзакции)
    bc.addBlock("Ivan -> Peter 5 BTC");
    bc.addBlock("Peter -> Maria 2 BTC");
    bc.addBlock("Maria -> Alex 1 BTC");

    // показваме веригата
    bc.showChain();

    // проверка на целостта
    cout << "\nChain status: "
         << (bc.isValid() ? "VALID" : "INVALID")
         << endl;

    cout << "\nPress Enter to exit...";
    cin.get();   // ← това спира програмата да се затвори веднага

    return 0;
}
